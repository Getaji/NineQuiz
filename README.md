NineQuiz
========

9問の3択クイズを作ろうという企画で作ったもの。

最早9問3択などではなくただのクイズを実行するだけのものと化している

## 手順
1. Question（問題）を作成
1. コレクションに格納する
1. 必要ならQuizLogicで使う正解時のメッセージとかをQuizLogicDataに設定する
1. QuizLogicを作成してQuestionのコレクション（と必要なら上記のQuizLogicData）を渡す
1. QuizLogicをstart
1. わいわいきゃっきゃ
